# 🌍 InAmigos Foundation Website

A professional NGO awareness website developed for **InAmigos Foundation** using **HTML & CSS**.

This project was created to spread awareness about the NGO’s mission, projects, social impact, volunteering opportunities, and community initiatives.

---

# ✨ Features

✅ Responsive NGO Website  
✅ Professional Homepage  
✅ About NGO Section  
✅ Ongoing Projects Showcase  
✅ Social Impact Section  
✅ Real NGO Gallery  
✅ Contact Form  
✅ Social Media Integration  
✅ Floating WhatsApp Button  
✅ Floating Call Button  
✅ Google Maps Integration  
✅ Mobile Friendly Design

---

# 🚀 Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Maps Embed
